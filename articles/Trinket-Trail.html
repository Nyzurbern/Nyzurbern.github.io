Trinket Trail

Overview

Trinket Trail is a simple app that allows users to save and manage their trinkets. Users can add trinkets, each with unique descriptions, placed in different rooms for organisation. After adding a photo of a trinket, the app will cut out the background and the user can then view the trinket in a 2D room.

What was used

  • Language(s): Swift, SwiftUI
  • Framework(s): Vision, State Management
  • Tools: Xcode, Figma
  • Platform: iOS

My Contributions

  • Helped design the app's user interface and user experience.
  • Compiled all the pages my team created so that the app would function correctly.
  • Implemented the core feature of adding trinkets with background removal using the Vision framework.
  • Assisted in testing and debugging to ensure the app's stability and performance using my personal phone.

Features

  • Create and manage a collection of trinkets.
  • Upload photos of your trinkets and view them in a 2D room, with background removal.
  • Organize and view your trinket collection in different virtual rooms with colour customization.

What I learned

I learned to use basic Swift and SwiftUI to develop a simple app and to work with the Vision framework for background removal and image processing. I also gained experience in creating a user-friendly interface for managing a collection of trinkets and organizing them in different virtual rooms.
