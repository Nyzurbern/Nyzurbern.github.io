Warthogs

Overview

Warthogs is a mobile game for iOS that utilizes the camera and Machine Learning to detect finger joints. The game is of the street-fighter style with different planets as the maps where the player can use their hands to hover over the screen to hit bubbles that allow them to attack the enemy. If the player fails to do so within a certain time limit, the enemy gets to attack instead.

What was used

  • Language(s): Swift, SwiftUI
  • Framework(s): Vision, State Management
  • Tools: Xcode, GitHub, Figma
  • Platform: iOS

My Contributions

  • Helped design the app's user interface and user experience.
  • Implemented the core feature of detecting finger joints using the Vision framework.
  • Implemented some of the game logic and mechanics, including health and scoring..
  • Assisted in testing and debugging to ensure the app's stability and performance using my personal phone.

Features

  • Play as 2 different characters and engage in combat against an enemy.
  • Use your hand to control the character's attacks by hovering over the screen.
  • Each character has unique abilities and animations.

What I learned

I learned to use the vision framework more deeply and gained experience integrating an existing finger-joint detection system into an iOS project of my own using the camera in SwiftUI.
