AWS Accelerator

Overview

In 2024, I participated in the AWS Accelerator program, an 8-month endeavor where I learnt python with Numpy and Pandas, machine learning with BigML and some web development with Flask and Anvil. The program was designed to provide a comprehensive understanding of data science concepts and their practical applications using AWS services.

Challenges

Collaborative

Working with a team of students from different schools was a challenge. We had to learn to communicate effectively and work together to achieve our goals. Some members also weren't as useful as others like me. I remember how 2 of my groupmates were very much into games instead of contributing. It was difficult getting over hurdles like this but by being patient with each other, we were able to overcome these challenges.

Technical

Learning to work with Python and understanding its libraries like Numpy and Pandas, was challenging. It was quite different from what I had faced before as I had never worked with data science tools and text-based programming languages extensively before. It was also difficult to wrap my head around the data analysis and machine learning concepts though it did get easier as time went on.

What I learned

I learnt a lot in the AWS Accelerator program. I learnt to properly code in text-based languages like python, how databases and APIs worked with Anvil and Flask respectively, and how to work with data science & machine learning tools like Numpy, Pandas and BigML. Overall, the AWS Accelerator program was a good learning experience and I am grateful for the opportunity to have participated in it.
